# hendragunawan.com

Personal portfolio of **Hendra Gunawan** — Senior Software Engineer.

A single-page site with a dark, terminal-inspired aesthetic.

## Tech

- [Next.js 15](https://nextjs.org) (App Router) + React 19
- TypeScript
- Tailwind CSS v4
- [Motion](https://motion.dev) for animation

## Develop

Requires **Node 22+**.

```bash
npm install
npm run dev      # http://localhost:3000
```

## Production build

```bash
npm run build
npm start
```

## Deploy

Deploys cleanly on [Vercel](https://vercel.com) — import the repo (or run `npx vercel`),
then attach the `hendragunawan.com` domain.
